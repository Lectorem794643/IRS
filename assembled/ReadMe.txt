Для запуска на windows надо запустить файл Windows.bat
Для запуска на Linux или MacOs, сначала надо дать разрешение для запуска скрипта с помощью команды "chmod +x Linux.sh", а потом запустить скрипт.
Для запуска на Linux или MacOs, сначала надо дать разрешение для запуска скрипта с помощью команды "chmod +x MacOs.sh", а потом запустить скрипт.
